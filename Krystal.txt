1.) Write a Java program to swap the value of two int variables, if the first number is 10 and the second number is 10.
	Before swap : Num1 = 10 
				  Num2 = 100

	Answer swap : Num1 = 100
				: Num2 = 10

Answer: 
public class backToJava {

	public static void main(String[] args) {

		
		  int num1 = 10;
		  int num2 = 100;
		  System.out.println("Before swap: Num1= " + num1 + " Num2= " + num2);
		  
		  int temp = num1; 
		  num1= num2; 
		  num2 = temp;
		              
		  System.out.println("After swap: Num1= " + num1 + " Num2= " + num2);
	}

}

4.) Given a string, return a "rotated left 2" version where the first 2 chars are moved to the end. The string length will be at least 2.

left2("Hello") → "lloHe"
left2("java") → "vaja"
left2("Hi") → "Hi"

Answer: 
public String left2(String str) {

		if (str.length() >= 2) {
			String firstTwo = str.substring(0, 2);
			String rest = str.substring(2);
			return rest + firstTwo;
		} else {
			return str;
		}
	}


5.) Ask user to enter 2 numbers ( use Scanner), and then tell them which number of the two is bigger

Answer: 
import java.util.Scanner;

public class backToJava {

	public static void main(String[] args) {
		
		Scanner scan = new Scanner(System.in);
		System.out.println("Please enter 2 numbers: ");
		int num1 = scan.nextInt();
		int num2 = scan.nextInt();
		scan.close();

		if (num1 == num2) {
			System.out.println("These two numbers are the same");
		} else {
			if (num1 > num2) {
				System.out.println(num1 + " is bigger");
			} else {
				System.out.println(num2 + " is bigger");
			}
		}
	}
}



6.) Given a string, we'll say that the front is the first 3 chars of the string. If the string length is less than 3, the front is whatever is there. Return a new string which is 3 copies of the front.

front3("Java") → "JavJavJav"
front3("Chocolate") → "ChoChoCho"
front3("abc") → "abcabcabc

Answer: 
public String front3(String str) 

	public String front3(String str) {

		String front = "";
		if (str.length() >= 3) {
			front = str.substring(0, 3);
		} else {
			front = str;
		}

		String result = front + front + front;
		return result;
	}
